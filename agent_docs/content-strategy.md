## Конкурентные преимущества над SimpleGate

Не упоминать конкурента напрямую, но закрывать его слабости:
- «5 способов идентификации» (vs 2 у конкурента: только PIN + BLE)
- «Именные интеграции: Trassir, Sigur, Ivideon» (vs только Wiegand)
- «Вертикальные решения для ЖК, КП, БЦ» (vs универсальный продукт)
- «Гибкие временные окна пропусков» (vs жёсткая привязка к полуночи)

## SEO-настройки (Rank Math Pro — для каждой страницы)

- Focus keyword: основной ключ из кластера
- SEO Title: `{Ключ} | PASS24.online — Облачная СКУД`
- Meta Description: 150–160 символов с ключом и CTA
- Schema: `SoftwareApplication` для продуктов, `FAQ` для вопросов, `LocalBusiness` для контактов
- Open Graph: 1200×630 px, брендированное изображение
