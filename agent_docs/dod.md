# Критерии завершенности (DoD)

Минимальный DoD для любой задачи:

- [ ] Функционал работает согласно требованиям
- [ ] Тесты проходят; добавлены новые при необходимости (или указано почему нет)
- [ ] Документация обновлена, включая `agent_docs/development-history.md`

Дополнительно по типам задач (если применимо):

Новая фича:
- [ ] Обновлен CHANGELOG (если ведется)
- [ ] Обновлена документация пользователя

Багфикс:
- [ ] Выявлена root cause
- [ ] Добавлен регрессионный тест

Рефакторинг:
- [ ] Функциональность не изменена (тесты подтверждают)
- [ ] ADR создан при значимых изменениях

Документация:
- [ ] Информация актуальна
- [ ] Примеры кода добавлены
- [ ] Ссылки работают
