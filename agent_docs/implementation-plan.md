# План реализации — 6 фаз

> Синхронизировано с `agent_docs/blueprint.md` v2.0 и `agent_docs/technical-spec.md` (февраль 2026)

## Фаза 1 — Фундамент (Недели 1–2)

### 1.1 Установка WordPress
- [ ] Чистая установка WordPress 6.5+ на хостинг (Kinsta / Cloudways)
- [ ] Настройка PHP 8.2, MySQL 8, Redis object caching
- [ ] SSL-сертификат, редирект HTTP → HTTPS, www → без www
- [ ] Настройка wp-config.php: `WP_DEBUG = false`, `DISALLOW_FILE_EDIT = true`, уникальные префиксы таблиц
- [ ] Удаление стандартных тем (Twenty*) и плагинов (Hello Dolly, Akismet — если не нужен)
- [ ] Настройка постоянных ссылок: `/post-name/`

### 1.2 Тема и конструктор
- [ ] Установка GeneratePress (бесплатная версия)
- [ ] Создание дочерней темы `pass24-child`
- [ ] Установка и активация Bricks Builder
- [ ] Создание глобальной дизайн-системы в Bricks:
  - Цветовая палитра (CSS-переменные из `architecture.md`)
  - Типографика Inter (Google Fonts или локально)
  - Глобальные классы кнопок (primary, secondary, ghost)
  - Шаблоны карточек
  - Бейджи доверия

### 1.3 Плагины — установка и настройка
- [ ] WP Rocket — включить кэш страниц, Gzip, lazy load картинок
- [ ] ShortPixel — режим WebP, агрессивное сжатие
- [ ] Rank Math Pro — подключить Google Search Console, настроить Schema
- [ ] Gravity Forms + Flamix Bitrix24 — создать вебхук в Битрикс24
- [ ] UpdraftPlus — ежедневный бэкап в облачное хранилище
- [ ] Wordfence — включить файрвол, настроить уведомления
- [ ] WP Mail SMTP — настроить через SMTP или Postmark/SendGrid

### 1.4 Аналитика
- [ ] Создать GA4 property, добавить через Google Site Kit
- [ ] Установить Яндекс.Метрику, включить Вебвизор + аналитику форм
- [ ] Google Tag Manager — контейнер и базовые теги
- [ ] Настроить цели GA4: `generate_lead`, `demo_request`, `calculator_used`
- [ ] Настроить зеркальные цели в Яндекс.Метрике

### 1.5 Bitrix24 — базовая интеграция
- [ ] Создать Inbound Webhook в Битрикс24 (`crm.lead.add`)
- [ ] Настроить плагин Flamix: поля маппинга (имя, телефон, email, тип объекта, UTM)
- [ ] Проверить передачу тестового лида
- [ ] Настроить поле `UF_CRM_LEAD_SCORE` (Integer) для скоринга
- [ ] Настроить поле `UF_CRM_REGION` для гео-маршрутизации

### 1.6 Cloudflare
- [ ] Подключить домен к Cloudflare
- [ ] Включить режим Proxied, HTTP/3, Brotli
- [ ] Page Rules: кэш статики, HTTPS-редирект
- [ ] Настроить Security Level: Medium

### 1.7 Подготовка контента (параллельно)
- [ ] Запросить согласие на использование логотипов у топ-клиентов (МТС, MLP, Sezar Group, Садовые Кварталы, Агаларов Эстейт, Росинка, Live Арена Амфион)
- [ ] Собрать скриншоты / видео интерфейса продукта для всех страниц
- [ ] Подготовить сканы дипломов наград в высоком качестве (Новатор Москвы 2022, ComNews Awards 2020, Безопасность 2023)
- [ ] Запросить отзывы у клиентов для секции 9 главной
