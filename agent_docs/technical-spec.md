# PASS24.online — Техническое ТЗ

> **Назначение:** Техническая спецификация для разработчика сайта PASS24.online
> **Платформа:** WordPress 6.5+
> **Версия:** 2.0
> **Дата:** Февраль 2026
> **Связанный документ:** `agent_docs/blueprint.md` (маркетинговый blueprint)

## 1. Технический стек

### Тема и конструктор страниц

**Основной вариант: Bricks Builder ($79, пожизненная лицензия) + GeneratePress**

- Самый чистый код среди WordPress-конструкторов
- Лучшие показатели Core Web Vitals «из коробки»
- Критично для ранжирования в Яндексе и Google

### Обязательные плагины

**Производительность:**

- WP Rocket ($59/год) — кэширование, минификация, оптимизация БД
- ShortPixel — сжатие изображений в WebP/AVIF
- Asset CleanUp — удаление ненужных скриптов на страницах
- Perfmatters — удаление лишнего кода

**SEO:**

- Rank Math Pro — поддержка API, AI-функции, Schema разметка
- Schema Pro — структурированные данные SoftwareApplication и FAQ

**Формы и захват лидов:**

- Gravity Forms ($59/год) + Flamix Bitrix24 интеграция
- Поддержка многошаговых форм, условной логики, автозахвата UTM-параметров
- Альтернатива: WPForms Pro + Flamix коннектор

**Аналитика:**

- Google Site Kit для GA4
- Яндекс.Метрика плагин (Вебвизор, тепловые карты, аналитика форм)
- Google Tag Manager для централизованного трекинга

**Онлайн-чат:**

- JivoSite с нативной интеграцией Битрикс24 из маркетплейса
- Поддержка WhatsApp, VK, Telegram каналов
- Альтернатива: нативный Live Chat Битрикс24

**Оптимизация конверсии (CRO):**

- If-So Dynamic Content — персонализация контента по сегментам
- Nelio A/B Testing — сплит-тестирование
- OptiMonk — попапы и exit-intent триггеры

**Безопасность:**

- Wordfence — файрвол + сканер
- UpdraftPlus — автоматические бэкапы
- WP Mail SMTP — надёжная доставка email

### Хостинг и производительность

- Хостинг: **Kinsta** или **Cloudways** (DigitalOcean/Vultr) с Redis object caching
- CDN: **Cloudflare** (бесплатный план)
- Целевые Web Vitals: LCP < 2.5 сек, INP < 200 мс, CLS < 0.1
- Активных плагинов: не более 20
- Все изображения: WebP с lazy loading ниже первого экрана

## 2. Интеграция с Битрикс24

### Конвейер данных WordPress → Битрикс24

```
┌──────────────────────────────────────────────────┐
│  САЙТ WORDPRESS                                   │
│                                                    │
│  Gravity Forms + плагин Flamix Bitrix24            │
│  Виджет чата JivoSite                             │
│  Нативные CRM-формы Битрикс24                     │
│  Калькулятор ROI (кастомный JS → webhook)          │
│                                                    │
│           │  webhook POST                          │
│           ▼                                        │
│  Битрикс24 Inbound Webhook                        │
│  POST https://{domain}/rest/{id}/{key}/            │
│       crm.lead.add.json                            │
└────────────────────┬─────────────────────────────┘
                     ▼
┌──────────────────────────────────────────────────┐
│  БИТРИКС24 CRM                                    │
│                                                    │
│  Лид создан → Автоскоринг → Маршрутизация         │
│                                                    │
│  Скор > 70 → Конвертация в Сделку → Менеджер      │
│  Скор 30-70 → Nurture-цепочка писем               │
│  Скор < 30 → Маркетинговая автоматизация           │
│                                                    │
│  Воронка: Новый → Квалифицирован → Демо →          │
│           КП → Переговоры → Выигран/Проигран       │
└──────────────────────────────────────────────────┘
```

### Модель скоринга лидов

| Поведение | Баллы | Обоснование |
|---|---|---|
| Посещение страницы тарифов | +15 | Высокий интент к покупке |
| Отправка запроса демо | +30 | Готов к продаже |
| Использование калькулятора ROI | +20 | Активная оценка |
| Скачивание спецификации/PDF | +20 | Техническая оценка |
| Просмотр 3+ страниц продуктов | +10 | Активное исследование |
| Возврат на сайт в течение 7 дней | +10 | Продолжающийся интерес |
| Просмотр видео >50% | +10 | Глубокое вовлечение |
| Тип компании = Застройщик/УК | +15 | Соответствие ICP |
| Email-домен = gmail/mail.ru | -5 | Возможно, не B2B |

**Реализация:** Поле Битрикс24 `UF_CRM_LEAD_SCORE` (Integer). Обновляется через n8n-воркфлоу на основе событий GA4 + целей Яндекс.Метрики.

### Квалификационный чат-бот JivoSite

```
[Посетитель заходит] → Контекстный триггер:

  На /products/* → «Какой тип объекта вы управляете?»
    ├─ ЖК → Воронка для застройщиков
    ├─ КП → Воронка для коттеджных посёлков
    ├─ БЦ → Коммерческая воронка
    └─ Другое → Общая квалификация

  На /pricing/ → «Сколько точек доступа вам нужно?»

  На /demo/ → Мгновенная запись через календарь Битрикс24

→ Все данные чата → Лид в CRM с полной историей переписки
```

### Автопубликация контента через WordPress REST API

```
n8n Workflow (запуск по расписанию):

  Триггер: Еженедельный cron

  → Шаг 1: API Claude/GPT → генерация статьи
    (тема из SEO-календаря, русский язык, 1500-2000 слов, HTML)

  → Шаг 2: API Pexels → подбор релевантного изображения

  → Шаг 3: POST /wp/v2/media (загрузка featured image)

  → Шаг 4: POST /wp/v2/posts
    {
      "title": "Сгенерированный заголовок",
      "content": "<html-контент>",
      "status": "draft",
      "categories": [id_категории],
      "featured_media": id_медиа,
      "meta": {
        "rank_math_focus_keyword": "целевой ключ",
        "rank_math_description": "SEO мета-описание"
      }
    }

  → Шаг 5: Уведомление контент-менеджеру в Битрикс24 на ревью
```

**Аутентификация WordPress REST API:** Application Passwords (Пользователи → Редактировать → Пароли приложений). Заголовок: `Authorization: Basic base64(username:app_password)`.

### Nurture-цепочка email

```
День 0:   Приветственное письмо + обзор + «Что такое облачная СКУД»
День 3:   Обучение: «5 признаков, что вашему объекту нужна модернизация СКУД»
День 7:   Кейс, подходящий сегменту лида (авто-выбор по полю Битрикс24)
День 10:  Сравнительный гайд + CTA на демо
          └─ Если кликнул на демо → мгновенное уведомление менеджеру
День 14:  Приглашение в калькулятор ROI + видео-отзыв
День 21:  Спецпредложение / сезонная акция
День 30:  Реактивация или архивация
```

**Реализация:** Нативный email-маркетинг Битрикс24 или **Unisender** (нативное приложение в маркетплейсе Битрикс24).

## 3. Аналитика и метрики

### Дуальный трекинг GA4 + Яндекс.Метрика

**События GA4:**

- `generate_lead` — все отправки форм
- `demo_request` — запрос демо
- `calculator_used` — использование калькулятора
- `file_download` — скачивание спецификаций
- `video_engagement` — просмотр видео
- `chat_initiated` — начало чата
- `scroll_depth_75` — глубина прокрутки

**Цели Яндекс.Метрики:**

- Зеркалировать все события GA4
- Включить Вебвизор для записи сессий
- Включить аналитику форм для отслеживания отказов по полям
- Настроить импорт офлайн-конверсий из Битрикс24 для ROI-атрибуции

### Структура UTM-меток

```
?utm_source={платформа}&utm_medium={канал}&utm_campaign={кампания}

Яндекс.Директ: utm_source=yandex&utm_medium=cpc&utm_campaign=skud_zhk_msk
VK Реклама:    utm_source=vk&utm_medium=social&utm_campaign=bc_access_control
Email:         utm_source=email&utm_medium=newsletter&utm_campaign=q1_2026
Telegram:      utm_source=telegram&utm_medium=social&utm_campaign=blog_post
```

> Плагины Flamix автоматически захватывают UTM-параметры из URL и передают в лиды CRM.

## 4. Техническая дорожная карта

### Фаза 1 (Недели 1-2): Фундамент

- Установка WordPress на Kinsta/Cloudways
- Настройка Bricks Builder + GeneratePress
- Установка всех плагинов
- Подключение GA4 + Яндекс.Метрика
- Настройка вебхуков Битрикс24 + Flamix
- Создание дизайн-системы и библиотеки компонентов

### Фаза 2 (Недели 2-4): Основные страницы

- Вёрстка главной (12 секций)
- Страница тарифов
- Лендинг запроса демо с многошаговой формой
- Страницы «О компании» и контактов

### Фаза 3 (Недели 4-6): Продукты и решения

- 6 страниц продуктов
- 8 вертикальных страниц решений
- Страница интеграций

### Фаза 4 (Недели 6-8): Инструменты конверсии

- Калькулятор ROI (кастомный JavaScript/React)
- Конфигуратор решений (wizard)
- Шаблон кейсов + 3-5 начальных кейсов
- Блог + 5-10 SEO-статей

### Фаза 5 (Недели 8-10): Автоматизация

- JivoSite с квалификационными потоками
- If-So персонализация
- Nurture-цепочки в Битрикс24/Unisender
- n8n автопубликация
- A/B-тестирование (Nelio)

### Фаза 6 (Постоянно): Оптимизация

- Еженедельные A/B-тесты
- Ежемесячная публикация SEO-контента (2-4 статьи)
- Ежеквартальные кейсы
- Анализ воронки через GA4 + Вебвизор
- Уточнение модели скоринга

## 5. Перспектива: расширение на MENA

Архитектура должна поддерживать:

- **Мультиязычность:** WPML или TranslatePress для арабского (RTL) и английского
- **Региональные лендинги:** `/en/solutions/uae-residential/`, `/ar/solutions/saudi-commercial/`
- **Маршрутизация CRM:** Поле `UF_CRM_REGION` по гео-IP (If-So), MENA-лиды в отдельную воронку
- **CDN:** Cloudflare edge-ноды в Дубае, Эр-Рияде, Каире для загрузки < 2 сек
