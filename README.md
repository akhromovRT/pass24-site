# 2030AI Project Template

Шаблон проекта для работы с Claude Code и Cursor.

## Важно

Перед началом работы заполните блок "Описание проекта и целей пользователя" в `AGENTS.md` — это основной контекст для агента.

## Структура документации

- `AGENTS.md` — универсальные правила, описание проекта и карта документов
- `agent_docs/index.md` — навигация по документам и шаблонам
- `agent_docs/architecture.md` — описание архитектуры (создается при необходимости)
- `agent_docs/adr.md` — архитектурные решения (создается при необходимости)
- `agent_docs/development-history.md` — история разработки (создается при необходимости)
- `task-context.md` — контекст между сессиями (создается по необходимости из `agent_docs/templates/task-context.md`)
- `agent_docs/` — все проектные документы, инструкции и шаблоны (progressive disclosure)

## Быстрый старт

1. Клонируйте репозиторий
2. Заполните блок описания проекта в `AGENTS.md`
3. Ознакомьтесь с картой документов в `AGENTS.md`
4. Начните разработку

## Документация

Подробные правила работы и стандарты проекта описаны в `AGENTS.md`.
